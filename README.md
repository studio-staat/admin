# Admin framework

A admin environment.

# Install

```sh
$ git clone https://github.com/studio-staat/admin myAdminApp
$ cd myAdminApp
$ npm i
```

# Development server

```sh
$ npm run dev
```

# Build

```sh
$ npm run build
```

# Start

```sh
$ npm start
```

# Deploy

> npm run build & npm run start

```sh
$ npm run deploy
```
